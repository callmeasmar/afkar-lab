# ADR-0002: Nom du projet — Afkar

**Statut**: accepted
**Date**: 2026-07-07

## Contexte

Le projet a besoin d'un nom qui reflète sa nature : un espace où les idées
naissent, se documentent et se transforment en création, pas juste un
assistant ou un outil de plus.

## Décision

Le projet s'appelle **Afkar** (أفكار), mot arabe pour "pensées / idées /
concepts".

## Alternatives considérées

- Un nom technique générique (ex: "PersonalOS") — rejeté : trop froid, ne
  reflète pas la dimension "idées avant tout".
- Un acronyme — rejeté : perd le sens en se voulant clever.

## Conséquences

- Le nom porte la philosophie du projet dès le premier contact.
- Afkar Lab / Afkar Core dérivent naturellement de ce nom.
