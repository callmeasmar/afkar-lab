# Project Charter — Afkar

## Vision

Créer un Personal Operations System capable d'orchestrer connaissances,
projets, recherches, automatisations, agents et infrastructure, dans un
flux unique et cohérent plutôt qu'en outils dispersés.

## Principes

1. **Compréhension avant automatisation** — ne pas automatiser ce qu'on
   ne comprend pas encore.
2. **Documentation avant construction** — écrire avant de coder.
3. **Simplicité avant complexité** — le plus simple qui marche.
4. **Modularité** — composants faiblement couplés.
5. **Apprentissage permanent** — chaque brique est une occasion d'apprendre.
6. **Composants remplaçables** — rien n'est figé, tout peut être remplacé
   si un meilleur outil/pattern émerge.

## Portée

Ce charter couvre Afkar Lab (privé, expérimental). Afkar Core (public)
aura son propre charter dérivé, une fois le projet mature.
